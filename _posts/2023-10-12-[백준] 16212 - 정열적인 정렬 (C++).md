## 문제

<https://www.acmicpc.net/problem/16212>

## 풀이

최대가 500,000이므로 $$ O(n^2) $$의 시간 복잡도를 가지는 알고리즘은 사용할 수 없습니다. 하지만 대부분의 정렬 함수는 시간 복잡도가 $$ O(n\log n) $$이므로, 함수만 사용하여 쉽게 해결할 수 있습니다.

```cpp
#include <algorithm>
#include <iostream>
#include <vector>
using namespace std;

int main() {
  int N;
  cin >> N;

  vector<int> a(N);
  for (int i = 0; i < N; i++) {
    cin >> a[i];
  }
  sort(a.begin(), a.end());

  for (int i = 0; i < N; i++) {
    cout << a[i] << " ";
  }

  return 0;
}
```
